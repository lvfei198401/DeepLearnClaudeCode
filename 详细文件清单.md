## 仓库结构

```
├── package/                    # 原始 npm 发布包
│   ├── cli.js                  # 打包后的单文件入口
│   ├── cli.js.map              # Source map（还原源码的来源）
│   ├── package.json            # 包配置 (v2.1.88)
│   ├── sdk-tools.d.ts          # SDK 工具类型定义
│   └── vendor/                 # 内嵌的原生二进制
│       ├── ripgrep/            # rg 搜索工具（多平台）
│       └── audio-capture/      # 音频捕获（多平台）
├── extract-sources.js          # 从 source map 提取源码的脚本
├── count_lines.py              # 代码行数统计脚本
├── restored-src/               # 还原出的源码（4756 文件，1902 个源文件，约 51 万行）
│   ├── node_modules/           # 还原出的依赖源码
│   └── src/                    # Claude Code 主应用源码 ↓↓↓
└── README.md
```
